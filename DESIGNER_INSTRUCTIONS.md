## 2. Вставка в Tilda Zero Block (пошагово)

### Шаг 1. Создать страницу в Tilda
1. Зайди в нужный сайт в Tilda.
2. Создай новую страницу или открой существующую.
3. Убедись, что страница в режиме редактирования.

### Шаг 2. Добавить блок HTML-код (Zero Block)
1. Нажми **«Добавить блок»** (или «+»).
2. Выбери раздел **«Другое»** (или «Дополнительно»).
3. Выбери блок **«HTML-код»** (Zero Block, иногда называется T123 или «Вставка HTML»).
4. Блок появится на странице — открой его для редактирования (двойной клик или кнопка «Настройки»).

### Шаг 3. Скопировать код из index.html
1. Открой файл **index.html** из архива в любом редакторе кода или текстовом редакторе (Notepad++, Sublime, VS Code, Блокнот).
2. Выдели **весь код** от самого начала тега **`<style>`** до самого конца тега **`</script>`** (включительно).  
   То есть: начало — строка с `<style>`, конец — строка с `</script>` перед `</body>`.
3. Скопируй выделенный фрагмент (Ctrl+C / Cmd+C).
4. В Tilda в блоке «HTML-код» удали весь placeholder-текст и **вставь** скопированный код (Ctrl+V / Cmd+V).
5. Сохрани блок и страницу.

### Шаг 4. Проверить отображение
1. Сохрани страницу и нажми **«Просмотр»** или открой страницу в новой вкладке.
2. Проверь: шрифты (Nunito, Playfair Display), даты (3–5 июля 2026), тарифы, контакты, что ничего не «наезжает» и текст читается.

### Шаг 5. Подключить формы и оплату (по желанию заказчика)
- Кнопки **«Забронировать»** и **«Выбрать»** сейчас ведут на якорь `#pricing`. В Tilda можно привязать их к форме записи или к оплате (настройки кнопки → ссылка на форму / оплату).
- Форма подписки на email в блоке рассылки — подключить через Tilda (форма подписки или свой сервис).

## 4. Если блок не вставляется целиком

В Tilda иногда есть лимит на размер блока. Тогда:
- **Вариант А:** разбить макет на 2–3 блока HTML-код: например, 1-й блок — от `<style>` до конца секции «Учителя», 2-й блок — расписание + тарифы + FAQ, 3-й блок — место + галерея + футер. В каждый блок копировать свой фрагмент **вместе с нужными стилями** (скопировать общий `<style>...</style>` в начало каждого блока или вынести общие стили в настройки сайта → HTML для всех страниц → CSS).
- **Вариант Б:** использовать Tilda Zero Block по инструкции из папки **TILDA_ZEROBLOCK** (разбивка по секциям).
